# Terminal Commands

## Git — الأهم
git init          → يحول المجلد لمشروع Git
git add .         → يجهز كل التغييرات للحفظ
git commit -m ""  → يحفظ التغييرات مع وصف
git push          → يرفع الكود لـ GitHub
git status        → يعرض حالة الملفات المعدلة
git log           → يعرض سجل كل التغييرات السابقة
git clone رابط    → ينسخ مشروع من GitHub لجهازك

## التنقل — مهم جداً
cd اسم-المجلد     → تدخل إلى مجلد
cd ..             → ترجع للمجلد السابق
pwd               → يخبرك أين أنت الآن

## إنشاء وحذف — مهم
mkdir اسم-المجلد  → ينشئ مجلداً جديداً
New-Item اسم.html → ينشئ ملفاً جديداً على Windows
Remove-Item اسم   → يحذف ملفاً على Windows

## عرض المحتوى
dir               → يعرض الملفات والمجلدات
clear             → ينظف شاشة Terminal

## VS Code
code .            → يفتح VS Code في المجلد الحالي
code اسم-الملف    → يفتح ملفاً معيناً
