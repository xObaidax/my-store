## HTML Boilerplate
### هيكل كل موقع الأساسي

<!DOCTYPE html>
<html lang="en">

    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>اسم الموقع</title>
    </head>

    <body>

        <header>
        <!-- header = قسم أعلى الصفحة — الشعار والبحث والسلة -->
        </header>

        <main>
    <!-- main = القسم الرئيسي الذي يحتوي على محتوى الصفحة -->
    <!-- يحتوي على المنتجات والعروض والتصنيفات -->
    </main>


        <footer>
        <!-- footer = الشريط الذي يظهر في أسفل كل صفحة -->
          <!-- يحتوي على معلومات التواصل وروابط مهمة -->
     </footer>

    </body>

</html>

---

## وسوم لا تحتاج إغلاق
meta, link, img, br, hr, input, area, col, source

---

## قواعد أساسية
- أسماء الملفات والمجلدات = أحرف صغيرة دائماً
- HTML تعليق = <!-- -->
- CSS و JS تعليق = /* */
- Terminal = بدون تعليق
- كل وسم يحتاج إغلاق إلا الـ 9 المذكورين أعلاه
